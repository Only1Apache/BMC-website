BMC — Functions Probe

  1. Open /.netlify/functions/hello — should return JSON { ok: true }
  2. Then open /.netlify/functions/get-posts — should return your posts JSON

If step 1 returns HTML, your deploy isn’t bundling functions or a UI redirect is overriding them.

If step 1 works but step 2 fails with 500, set env vars NETLIFY_TOKEN and SITE_ID/WEB_ID and redeploy.
